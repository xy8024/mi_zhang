# 前端社招笔试题

## 1.广告的可见率

```
1.Active View由哪⼏个因素决定？
	答：Active View由两个重要因素，
	（1）.广告的哪一部分出现在屏幕上的可见空间中
	（2）.广告的这一部分保持可见的时间

2.Active View(⼴告可⻅率)和Impression(⼴告展示) 这两个指标有什么区别？
	答：Active View可见性有助于确定用户实际看到广告的可能性。如果一个广告至少有50%的面积在屏幕上显示了至少一秒钟(根据互动广告局(IAB)的标准，这是最低标准)，则该广告被视为可观看的
	Impressions要考虑在网页底部的位置。如果用户需要向下滚动才能看到页面底部，那么在导航离开页面之前，用户可能看不到广告。然而，对于屏幕较大的用户来说，整个页面可能出现在一个可见空间中——这意味着广告也是可见的。这两种情况都可以算作⼴告展示

3.对于⼀个主要⾯向移动设备的⽹站(90%以上⽤户使⽤⼿机访问)，可能提⾼⼴告的可⻅率有哪些⽅法?
	答：调整广告展示位置。通过分析网站流量统计数据，了解访客所用上网设备及操作系统，然后，尽量将广告放置在首屏靠下的位置。此位置的可见率最高。此外，顶部导航栏下方的广告比导航栏上方的广告具有更高的可见率。应使用自适应广告，以使所有的上网设备上展示最佳的广告效果。

```

## 2.网页性能指标

```
1. LCP指标是指什么？健康的LCP指标是⼀个什么样的标准？
	答：LCP指标指的是最大内容绘制(LCP)指标报告viewport中可见的最大图像或文本块的渲染时间，相对于页面第一次开始加载的时间。
	健康的LCP指标是为了提供良好的用户体验，网站应该努力将最大内容绘制时间控制在2.5秒或更短。为了确保大多数用户都能达到这一目标，一个很好的阈值是页面负载的75%，包括移动设备和桌面设备。


2.除了LCP⽂中还提到了那些指标，请列出3个，并解释他们的不⾜之处。
	答：load、DOMContentLoaded，不足：load或DOMContentLoaded不一定与用户在屏幕上看到的内容相对应。
	FMP和SI指标，不足：这些指标非常复杂，难以解释，而且经常是错误的——这意味着它们仍然无法识别页面的主要内容何时已加载。


3.根据⽂中对LCP的描述，在开发Web⽹站时，有哪些提⾼LCP的措施？
	答：网站应该努力将最大内容绘制时间控制在2.5秒或更短。为了确保大多数用户都能达到这一目标，一个很好的阈值是页面负载的75%，包括移动设备和桌面设备。
```
